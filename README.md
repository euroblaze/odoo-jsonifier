# JSONifier

Odoo 18 module for JSON serialization of any model data.

## Description

Export any Odoo model data to JSON format with configurable field mappings.

## Features

- JSON export for all models
- Configurable field resolvers
- Flexible data transformation

## Requirements

- Odoo 18.0
- base module

## Installation

1. Download and extract to your Odoo addons folder
2. Update apps list in Odoo
3. Install "JSONifier"

## Configuration

Configure JSON exports via Settings > Technical > Exports.
